# Expense Management App

## Overview
This is a web application for managing daily expenses.  
Users can easily record, view, and manage their spending with a simple and intuitive interface.

## Features
- Add, edit, and delete expense records
- Display expense list
- Simple and user-friendly UI
- Responsive design (if applicable)

## Tech Stack
- Frontend: React, TypeScript, HTML, CSS
- Backend: Node.js
- Database: SQLite3

## Architecture / Design
- Designed with separation between frontend and backend
- Focused on clean and maintainable code structure
- Implemented efficient data handling for performance and scalability

## Key Points / What I focused on
- Designed UI/UX for easy input and readability
- Structured data for better management and future expansion
- Wrote code with maintainability and clarity in mind

## Challenges
- Managing state efficiently in React
- Designing a simple but scalable data structure

## Future Improvements
- Add data visualization (charts/graphs)
- Implement file import/export (CSV, etc.)
- Improve performance and refactor code

1. Clone the repository
2. Install dependencies
3. Run the application
